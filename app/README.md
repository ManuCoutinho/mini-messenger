## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/ManuCoutinho/mini-messenger

# Access
$ cd app

# Install dependencies
$ npm install

# Run the project
$ npm start

# The server will initialize in the <http://localhost:5173>
```
